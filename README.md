# 💰 Finans

Projeto desenvolvido durante o curso **Desenvolvimento Web Completo** da Udemy, com o objetivo de praticar a criação de landing pages modernas utilizando HTML5, CSS3 e Bootstrap.

## 📖 Sobre o Projeto

O Finans é uma landing page institucional criada para apresentar um serviço financeiro fictício.

O projeto foi desenvolvido com foco na construção de interfaces modernas, responsivas e organizadas, aplicando boas práticas de desenvolvimento front-end por meio da utilização de HTML semântico, CSS personalizado e recursos oferecidos pelo framework Bootstrap.

A proposta do projeto é simular uma página de apresentação de produto ou serviço, modelo amplamente utilizado por empresas, startups e plataformas digitais.
